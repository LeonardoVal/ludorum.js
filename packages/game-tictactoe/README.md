TicTacToe for Ludorum
=====================

[![NPM](https://nodei.co/npm/@ludorum/game-tictactoe.png?mini=true)](https://www.npmjs.com/package/@ludorum/game-tictatoe)

Ludorum is a board game framework. It is not focused on graphics or user interfaces, but on artificial players design, implementation and testing.

This package provides an implementation of the [TicTacToe game](http://en.wikipedia.org/wiki/Tictactoe), which is a very simple and traditional game.
