Ludorum is a board game framework. It is not focused on graphics or user interfaces, but on artificial players design, implementation and testing. Among other thing, it currently includes: * Players based on MiniMax with alfa-beta pruning, Max-N and pure Monte Carlo tree search. * Simple reference games, like: TicTacToe, ToadsAndFrogs, Mancala and ConnectFour.

It supports loading with AMD (with RequireJS) or a script tag (sets 'ludorum' in the global namespace). In order to work requires another library of mine called basis.

Development
